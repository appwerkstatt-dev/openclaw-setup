# Security Policy

## Supported Scope

Dieses Repo stellt Setup-Skripte und Betriebsdoku bereit.
Es gibt keine laufende Security-Garantie fuer alle Umgebungen.

## Responsible Disclosure

Bitte melde sicherheitsrelevante Themen nicht zuerst oeffentlich im Issue-Tracker.
Nutze stattdessen einen privaten Kanal (z. B. direkte Kontaktaufnahme mit dem Maintainer).

## Was gemeldet werden sollte

- Unsichere Defaults in Skripten
- Moegliche Privilege-Escalation-Pfade
- Secret-Leaks oder Logging sensitiver Werte
- Fehlkonfigurationen, die Exposition erhoehen

## Erwartung an Betreiber

- Eigene Threat-Analyse durchfuehren
- Regelmaessige Updates einplanen
- Zugangsschluessel rotieren
- Backup/Restore regelmaessig testen
